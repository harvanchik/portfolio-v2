Current Core Project

SvelteKit 2 | TypeScript | Cloudflare

playims.com

A next-generation intramural and tournament management platform replacing slow, frustrating league tools with fast, intuitive workflows for real operators.

  • Primary goals: simplify league setup, improve scheduler/tournament control, and reduce admin friction at scale.
  • Stack: SvelteKit 2, Svelte 5 runes, TypeScript 5, TailwindCSS 4, Cloudflare D1 (SQLite), Drizzle ORM, Zod, Cloudflare Pages.
  • Architecture: multi-tenant-ready data model, API-driven backend endpoints, migration-first database workflow, and dynamic client theming.
  • Product direction: schools, universities, municipalities, and private organizations running leagues and tournaments.
  • Long-term objective: ship a modern foundation for automation, analytics, and integrations in intramural operations.
